## &nbsp; `[ 03 ]` &nbsp; RULES I BUILD BY

<br/>

<div align="center">

| # | Rule | Because |
|---|------|---------|
| 01 | Understand before you implement | Cargo-culting code is just borrowing someone else's bugs |
| 02 | Name things like they matter | `x`, `temp`, `data` — these are surrenders, not variables |
| 03 | Slow is smooth. Smooth is fast. | Methodical > frantic every single time |
